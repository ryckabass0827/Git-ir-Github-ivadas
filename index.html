Terminalo komandos

  • git init - inicializuoja git repozitorija
  • ls/dir - isvardina aplankalus kurie yra dabartineje repozitorijoje
  • cd - change directory
  • TAB - skirtas uzbaigti arba uzpildyti terminalo komanda arba pateikti pasiulyma

git komandos

  • git init - inicijuoja git repozitorija
  • git status - parodo git repozitorijos statusa
  • git add - prideti failus prie paruostu comitinimui failu
  • git commit -m "zinute" - sukuriama kodo versija su zinute
  • git restore ./*/file - panaikina pries tai comitintus pakeitimus (kurie nera addinti)
  • git restore --staged ./*/file - isima nurodyta faila is staged zonos
  • git push - nusiuncia naujus comitus i serveri
  • git log - rodo git comitu istorija
  • git branch commitID - nukeliauja i nurodyta kodo versija
